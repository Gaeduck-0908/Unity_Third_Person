# Unity_ThirdPerson_JumpGame
(제작중)<br>
simple ThirdPerson JumpGame

# Control
 - W : Front move
 - A : Left move
 - D : Right move
 - S : Back move
 - Shift : Dash
 - Space bar : Jump
